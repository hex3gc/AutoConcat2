AutoConcat v2

A tool for processing strings of text and combining them with custom delimiters.


Overview

The program has three main sections:

  • The Sidebar where settings and the Project Picker can be accessed.
  • The Concatenation Area where the input text and delimiters are defined, and output is displayed.
  • The Operations Area where text manipulation operations can be applied to the concatenation process.

Sidebar

The top of the Sidebar gives you four options:

  • Settings, opening a menu where you can change how the program looks and works.
  • Help, for accessing this manual at any time.
  • New Project resets all the text and Operations to default, giving you a fresh project to work with.
  • Save Project As... allows you to create and name a new Project from the current text inputs, delimiters and Operations. This will be saved below in the Project Picker.

The Project Picker lets you save and load Projects. These represent different states of the program including its input and output text, Operations and delimiters. It can be useful to create a Project for commonly re-visited actions that you'll need to return to later as these will be saved even if you close the program. You can use the buttons on the bottom of the Project Picker to Save the current project using the data on-screen, to Load an existing project, or to Delete one from the picker.

The save file for your Projects and other application settings can be accessed by typing %appdata% into the bar at the top of your computer's File Explorer, then navigating to the AutoConcat2 folder.


Concatenation Area

This area contains input and output text boxes, between which are delimiter options. The functionality of this area is carried out in five steps:

  1. The input box is read from as a string of text.

  2. 'Pre-concat' Operations are applied to the string.

  3. All whitespace (Spaces, tabs, line breaks, etc.) characters are removed from the string and replaced with the infix delimiter. Then, the prefix and suffix are placed on the front and end of the string respectively.

  4. 'Post-concat' Operations are applied to the string.

  5. The resulting string is shown in the output box.

This process allows you to make a variety of changes to your text input, for example taking a column or row of values from Excel and transforming them into a list format readable by an SQL query. The input box accepts pasting from most software, and if the formatting of the input text is incorrect you can always use Operations to modify it to your needs.

Delimiter Presets persist between Projects and allow you to save a set of delimiters for later use. Some default ones are provided when the program starts up, but new ones can be defined or deleted using the buttons at the bottom of this area.


Operations

Operations are text modifiers that can be inserted into the concatenation process, and can be added by choosing a name from the dropdown menu and pressing the + button.

These execute from top to bottom, so using the up and down arrows to arrange the order of Operations may change how the text is processed. The Process After Concat checkbox determines whether this Operation will be run before or after the text goes into the Concatenator.

For example, all projects have a 'trim whitespace' Operation included by default to clean up the edges of the string and avoid infix delimiters being applied there. More information about each Operation is provided in their descriptions.
